Actions that the robot can perform

  1. Open/Close Object

    Opens the cupboard or fridge or some other object that can be opened. Example usage will be to open the cupboard to find milk inside it.

  2. Push object to destination:

    Sometimes an object may be too heavy to pick up for the robot. In such a case, the robot goes near the source object and pushes it to a place near the destination object.

  3. Climb up an object/ Climb down an object

    Sometimes an object may be at a different height than the robot, i.e the robot may not be able to reach it. In such a case, the robot can climb onto the object specified, so that its height may increase.
