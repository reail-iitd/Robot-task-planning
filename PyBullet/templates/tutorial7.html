Actions that the robot can perform

  1. Stick object on destination:

    This allows the robot to stick one object (source) on the other. As objects cannot be stuck directly, you may want to use the apply action to apply glue on the object to stick it. For example, sticking a paper to the wall using glue.

  2. Clean object:

    You can clean dirt using this action. You must have the object to be used for cleaning in your hand, when the clean action is called.
