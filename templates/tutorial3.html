Actions that the robot can perform

  1. Move robot to object:

    Moves the robot to the object specified.

  2. Pick source:

    Moves the robot to the source object, picks it up in its hand.

  3. Drop object on destination:

    To execute this action, the robot must have this object in its hand. The robot will drop the specified object in its hand, onto/into the destination object.

  4. Pick source and place on destination:

    Moves to the source object, picks it up, and then carries it to the destination object and places it on top of it. Example usage: to pick and place a book inside the box.
